        <h1 id="introduction">Introduction</h1>
<p><em>This is an introduction for how to work using the Arduino IDE for the experiments.</em></p>
<h3 id="software-">Software :</h3>
<p><em>We use Arduino IDE for writing,compiling and uploading the arduino codes. You can download the latest IDE from <a href="https://www.arduino.cc/en/software">here</a> for linux,windows and mac.</em></p>
<h2 id="1">1:</h2>
<p><em>The software interface would be like this</em> <br />
<img src="https://user-images.githubusercontent.com/86108610/148682385-360abc84-7c36-4309-ac6c-0f0632508f42.png" alt="interface" /></p>

<h2 id="2">2:</h2>
<p><em>You can just type the code,save and compile it in the IDE. To compile the program just click the tick button.</em> <br />
<img src="https://user-images.githubusercontent.com/86108610/148682544-972f7432-8e17-4a76-84eb-815d478e7e47.png" alt="Compilation" /></p>

<h2 id="3">3:</h2>
<p><em>Now connect the arduino board to the system with a USB cable and select the port(Tools-&gt;Port-&gt;Serial Port).</em>    <br />
<img src="https://user-images.githubusercontent.com/86108610/148682719-672d6d19-cfe0-4d5f-8c9c-566923f2b1b3.png" alt="Port" /> <br />
<em>Click the right arrow button to upload the code to the board.</em> <br />
<img src="https://user-images.githubusercontent.com/86108610/148682738-77d59bc9-15c8-401c-80b0-c0742fa93ffe.png" alt="Upload" /></p>

<p><em>The circuit will now work according to code uploaded.</em></p>

<h2 id="note">Note:</h2>
<p><em>For some experiments we use Serial Monitor to get output. For this click on the serial monitor button on the right side and you will get the output in a new window.</em> <br />
